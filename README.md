# Hangman Game 

## Project Description
This is a simple Hangman game developed using Python.  
The application is built with **Streamlit** for the user interface and **random** module to select words.

## Technologies Used
- Python
- Streamlit
- Random

## Features
- Random word selection
- Interactive user interface
- User-friendly gameplay
- Real-time input and output

## File Details
- hangman_app.py – Main application file

## How to Run the Project
1. Install required library:
   pip install streamlit

2. Run the application:
   streamlit run hangman_app.py

## Output
The game runs in a web browser where users can guess letters and play the Hangman game interactively.
